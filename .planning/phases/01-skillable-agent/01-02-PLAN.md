**完整流程:**

```python
import os
import logging
from skill_manager import SkillManager

# 1. 环境变量校验（per D-05: AGENT_SKILLS 驱动）
REQUIRED_VARS = ["AZURE_OPENAI_ENDPOINT", "AZURE_OPENAI_CHAT_DEPLOYMENT_NAME"]

# 2. 读取 Skill 配置
AGENT_SKILLS = os.environ.get("AGENT_SKILLS", "")
# 格式: "local:travel-advisor,clawhub:couple-coach,url:https://..."
# 空值表示不加载任何外部 skill，仅使用 base instructions

BASE_INSTRUCTIONS = os.environ.get(
    "AGENT_BASE_INSTRUCTIONS",
    "You are a helpful assistant enhanced with specialized skills."
)
MAX_SKILL_TOKENS = int(os.environ.get("AGENT_MAX_SKILL_TOKENS", "4000"))

# 3. 加载 Skill
sm = SkillManager()
skills = []

# 3a. 始终加载本地 skills/ 目录（如果没指定 AGENT_SKILLS）
# 3b. 如果指定了 AGENT_SKILLS，按声明加载
if AGENT_SKILLS:
    skill_specs = [s.strip() for s in AGENT_SKILLS.split(",") if s.strip()]
    skills = sm.load_skills(skill_specs)
else:
    # 默认加载内置 skills/ 目录下的所有 skill
    skills_dir = os.path.join(os.path.dirname(__file__), "skills")
    if os.path.isdir(skills_dir):
        skills = sm.load_from_directory(skills_dir)

# 4. 组合 instructions
instructions = sm.compose_instructions(BASE_INSTRUCTIONS, skills, max_tokens=MAX_SKILL_TOKENS)

# 5. 可选 Foundry Tools（web_search, MCP）
tools = []
if os.environ.get("ENABLE_WEB_SEARCH", "").lower() == "true":
    tools.append({"type": "web_search_preview"})
if conn_id := os.environ.get("AZURE_AI_PROJECT_TOOL_CONNECTION_ID"):
    tools.append({"type": "mcp", "project_connection_id": conn_id})

# 6. 创建 Agent（per D-01: Agent Framework ChatAgent）
from agent_framework.azure import AzureOpenAIChatClient
from azure.ai.agentserver.agentframework import from_agent_framework, FoundryToolsChatMiddleware
from azure.identity import DefaultAzureCredential

middleware = FoundryToolsChatMiddleware(tools) if tools else None
chat_client = AzureOpenAIChatClient(
    credential=DefaultAzureCredential(),
    middleware=middleware,
)
agent = chat_client.create_agent(
    name="SkillableAgent",
    instructions=instructions,
)

# 7. 启动服务
from_agent_framework(agent).run()
```

**日志输出:** 启动时打印已加载 Skill 清单和 token 统计，便于调试。

**关键点:**
- 导入 agent_framework 和 azure SDK 放在使用处（非模块级），避免环境变量未就绪时 import 失败（per Pitfall 7）
- middleware 仅在有 tools 时创建
- 默认行为：无 AGENT_SKILLS 时加载本地 skills/ 全部内容
