**1. Skill dataclass (per D-02, D-06, D-08) with agentskills.io fields:**
```python
@dataclass
class Skill:
    name: str              # Required: lowercase, hyphens, 1-64 chars
    description: str       # Required: 1-1024 chars
    content: str           # Markdown body (injected into instructions)
    source: str            # "local" | "git" | "clawhub" | "url"
    token_estimate: int    # len(content) // 4
    license: str = ""           # Optional
    compatibility: str = ""     # Optional
    metadata: dict = None       # Optional: arbitrary key-value
    allowed_tools: str = ""     # Optional: space-delimited tool list
    skill_dir: str = ""         # Path to skill dir (for scripts/, references/)
```

**2. SkillManager class:**

- `parse_skill_file(raw, source="local", skill_dir="") -> Skill`
  - Use python-frontmatter to parse YAML frontmatter + Markdown body (per D-03)
  - Extract agentskills.io fields: name, description, license, compatibility, metadata, allowed-tools
  - Fallback: name="unnamed", description="" for files without frontmatter
  - token_estimate = len(content) // 4

- `load_from_directory(path) -> list[Skill]`
  - Primary: scan for agentskills.io structure `path/*/SKILL.md` (each subdir is a skill)
  - Fallback: scan `path/*.md` flat files
  - Store skill_dir = subdirectory path for references/ access
  - Skip unparseable files with logged warning

- `load_from_git(repo_url, skill_names=None) -> list[Skill]` **(per D-07)**
  - Validate repo_url: must match `git@...` or `https://...` patterns via regex
  - Extract repo name from URL for cache dir
  - Clone to `/tmp/skills_cache/git/{repo_name}/` with `subprocess.run(["git", "clone", "--depth", "1", repo_url, target_dir], timeout=60)`
  - If cached dir exists, run `subprocess.run(["git", "-C", target_dir, "pull"], timeout=30)` to update
  - Scan cloned repo: check `skills/` subdir first, then root `*/SKILL.md`
  - If skill_names list provided, only load matching skills; otherwise load all
  - No shell=True in subprocess (security)

- `load_from_clawhub(slug) -> Skill`
  - Validate slug: `re.fullmatch(r'[a-zA-Z0-9_-]+', slug)` (security - prevent URL injection)
  - httpx GET to ClawHub API, timeout 30s
  - ZIP -> zipfile.ZipFile(io.BytesIO(response.content)) -> find SKILL.md
  - Cache to `/tmp/skills_cache/clawhub/{slug}/`
  - Raise SkillLoadError on failure

- `load_from_url(url) -> Skill`
  - Validate URL scheme: only http/https (security)
  - httpx GET, timeout 30s, max 1MB (security)
  - parse_skill_file(response.text, source="url")

- `load_skills(skill_specs: list[str]) -> list[Skill]`
  - Parse `"source:identifier"` format:
    - `"local:travel-advisor"` -> load from skills/ dir, filter by name
    - `"git:git@github.com:kepano/obsidian-skills.git"` -> load_from_git (all)
    - `"git:git@github.com:kepano/obsidian-skills.git#obsidian-markdown"` -> load_from_git with specific skill
    - `"git:git@github.com:kepano/obsidian-skills.git#obsidian-markdown,defuddle"` -> multiple specific skills
    - `"clawhub:couple-coach"` -> load_from_clawhub
    - `"url:https://example.com/SKILL.md"` -> load_from_url
  - Continue on individual failures, log errors

- `compose_instructions(base, skills, max_tokens=4000) -> str`
  - Format: base + "\n\n" + for each skill: `"== Skill: {name} ==\n{description}\n\n{content}"`
  - Log total token estimate, warn if over max_tokens
  - Return composed string

**Exception:** `class SkillLoadError(Exception)`

**Logging:** `logging.getLogger(__name__)`
