创建 `agent-framework/agent-with-skills/skill_manager.py`，包含:

**1. Skill dataclass（per D-02: Skill 是 prompt 扩展）:**
```python
@dataclass
class Skill:
    name: str           # 来自 frontmatter 或文件名
    description: str    # 来自 frontmatter 或空
    content: str        # Markdown body（注入 instructions 的内容）
    source: str         # "local" | "clawhub" | "url"
    token_estimate: int # len(content) // 4 粗略估算
```

**2. SkillManager 类，包含以下方法:**

- `parse_skill_file(raw: str, source: str = "local") -> Skill`
  - 使用 `python-frontmatter` 解析 YAML frontmatter + Markdown body（per D-03）
  - 兼容无 frontmatter 的纯 Markdown 文件（name 取 "unnamed"，description 取空）
  - token_estimate = len(content) // 4

- `load_from_directory(path: str) -> list[Skill]`
  - 使用 `pathlib.Path.glob("*.md")` 扫描目录
  - 对每个文件调用 `parse_skill_file()`
  - 跳过解析失败的文件（log warning）

- `load_from_clawhub(slug: str) -> Skill`
  - 使用 `httpx`（per D-04）GET `https://wry-manatee-359.convex.site/api/v1/download?slug={slug}`
  - 响应是 ZIP 文件 → 用 `zipfile.ZipFile(io.BytesIO(response.content))` 解压
  - 在 ZIP 中查找 `SKILL.md` 文件 → `parse_skill_file()`
  - 缓存到 `/tmp/skills_cache/{slug}/SKILL.md`，下次先检查缓存
  - 超时 30 秒，连接失败抛出 `SkillLoadError`

- `load_from_url(url: str) -> Skill`
  - 使用 `httpx` GET 下载 .md 文件
  - `parse_skill_file(response.text, source="url")`
  - 超时 30 秒

- `load_skills(skill_specs: list[str]) -> list[Skill]`
  - 解析格式: `"source:identifier"` (e.g., `"local:travel-advisor"`, `"clawhub:couple-coach"`, `"url:https://..."`)
  - 路由到对应的 load 方法
  - 返回所有成功加载的 Skill 列表

- `compose_instructions(base: str, skills: list[Skill], max_tokens: int = 4000) -> str`
  - 格式: `base + "\n\n" + 每个 skill 用 "--- Skill: {name} ---\n{content}" 分隔
  - 计算总 token：base_tokens + sum(skill.token_estimate)
  - 若超过 max_tokens，log warning（不截断，仅警告）
  - 返回组合后的完整 instructions 字符串

**自定义异常:**
- `class SkillLoadError(Exception)` — 加载失败

**日志:** 使用 `logging.getLogger(__name__)` 记录加载过程。

**安全注意事项:**
- ClawHub slug 参数校验：仅允许 `[a-zA-Z0-9_-]` 字符，防止 URL 注入
- URL 下载限制响应大小（最大 1MB），防止恶意大文件
- ZIP 解压仅提取 SKILL.md，忽略其他文件，防止 zip bomb
